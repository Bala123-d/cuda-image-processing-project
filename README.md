# CUDA Image Processing Project

## Project Description
This project demonstrates GPU computing using CUDA. 
The program processes image data using a CUDA kernel running on the GPU.

The goal is to show how GPU parallelism can accelerate image processing tasks.

## How to Compile

nvcc grayscale.cu -o processor

## How to Run

./processor

## Dataset
Multiple images are used as input data to demonstrate batch processing.

## GPU Processing
The CUDA kernel runs on the GPU and processes the data in parallel using many threads.
